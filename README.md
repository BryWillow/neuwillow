# neuwillow

NeuWillow is all about neuroscience, specifically how neurons communicate.
You can model action potentials, by specifying parameters in Configuration.json.
You can specify what neurotrnsmitters go into each pre-synaptic vesicles before going into the synapse.
Many properties of ion channels, proteins, etc. Doxygen and more functionality to come.
This project uses CMake: as you can see under src\build.
